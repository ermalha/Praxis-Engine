# TUI Keybinds Reference

## Global

| Key | Action |
|-----|--------|
| `1` | Switch to Work Queue screen |
| `2` | Switch to Conversation screen |
| `3` | Switch to Engagement screen |
| `4` | Switch to Audit screen |
| `q` | Quit the TUI |
| `?` | Show help overlay |
| `r` | Refresh current screen |
| `w` | Trigger a manual wake cycle |

## Work Queue Screen

| Key | Action |
|-----|--------|
| Up/Down | Navigate items |
| `r` | Refresh queue data |

## Conversation Screen

| Key | Action |
|-----|--------|
| Enter | Send message |
| `r` | Refresh |

## Engagement Screen

| Key | Action |
|-----|--------|
| Tab | Switch between tabs |
| `r` | Refresh all tabs |

## Audit Screen

| Key | Action |
|-----|--------|
| `r` | Refresh audit log |
