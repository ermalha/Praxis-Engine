# Weak Story Example

## Story

As a **user**, I want to **manage my account**.

## Problems

- **No "so that"** — No value clause, can't prioritize or negotiate
- **Not Independent** — "Manage" implies many features (create, edit, delete, settings)
- **Not Small** — This is an epic, not a story
- **Not Testable** — "Manage" is too vague for acceptance criteria
- **Role too generic** — "User" doesn't tell us which user type

## How to fix

Split into specific stories:

1. As a **registered user**, I want to **update my display name**, so that **my profile shows my preferred name**.
2. As a **registered user**, I want to **change my email address with verification**, so that **I can keep my contact info current**.
3. As a **registered user**, I want to **deactivate my account**, so that **my data is no longer processed**.

Each of these is Independent, Small, Testable, and Valuable.
